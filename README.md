# Git-Course-SOT
Repositorio de handson para SOT Netrix

Obejtivo del Repositorio:
1. La interaccion con comandos basico de git
2. Hands-on para comprender el flow del repo
3. Metodo de "evaluacion" del training

Enjoy Sotitos!

## Flujo de trabajo
- main: rama estable
- develop: rama de desarrollo
- feature/*: nuevas funcionalidades
- hotfix/*: correcciones urgentes

## proceso de instalacion
- Siempre que quiera armar una branch nueva debo salir de DEVOPS
